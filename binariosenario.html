sistema binario

Il sistemi di conteggio con le dita binario è probabilmente il sistema utilizzato che permette di raggiungere il numero (decimale) più elevato, viene utilizzato per contare ed effettuare operazioni matematiche in base 2. Il sistema binario come dice già il nome si basa su un sistema in base due in cui ogni cifra ha due possibili stati: 0 e 1. Ogni dito quindi se alzato rappresenta il valore 1 e se abbassato rappresenta il valore 0. Il valore decimale più alto comunicabile attraverso le 2 mani è il 1023 con tutte le dita sollevate.
